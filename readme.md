# UOS 멘토 검색

서울시립대학교 멘토링 데이터를 검색할 수 있는 웹 서비스입니다.
2025-12-04 기준 데이터를 바탕으로 멘토의 이름, 회사, 부서, 직무 등의 정보를 검색할 수 있습니다.

## 주요 기능

- **멘토 데이터 추출**: 원본 JSON 데이터에서 필요한 정보(이름, 소속, 직무 등)를 추출하여 단일 파일로 병합합니다.
- **데이터 조인**: 직무 코드(Code)를 읽기 쉬운 텍스트(Name)로 자동 변환합니다.
- **웹 검색 인터페이스**: 키워드(이름, 회사명, 부서명 등)를 통해 실시간으로 멘토를 검색할 수 있습니다.

## 실행 방법

### 1. 데이터 추출 (선택 사항)
`data/` 폴더 내의 JSON 파일들로부터 멘토 정보를 추출하여 `mentors_extracted.json`을 생성합니다.
(이미 `mentors_extracted.json`이 있다면 건너뛰어도 됩니다.)

```bash
python extract_mentors.py
```

### 2. 서버 실행
Python 내장 모듈을 사용하여 웹 서버를 실행합니다. 별도의 라이브러리 설치가 필요하지 않습니다.

```bash
python server.py
```

### 3. 접속
브라우저에서 아래 주소로 접속하여 검색을 이용할 수 있습니다.

[http://localhost:8000](http://localhost:8000)
